# Roles and Permissions

## Review roles
- **Internal staff** (Collection Managers, Curators): can edit, approve, and override review locks.
- **External experts** (External Experts): can edit and comment but cannot approve.

## Lock overrides
Administrators and Collection Managers can release or reassign locks in the admin interface and can override locks when opening pages.

## Notes
- Group membership is managed through Django groups.
- Approval permissions are enforced in the review views.
